#OhMyFood
<a name="readme-top"></a>
 
## Application :
Menu reservation application in restaurants

![My Image](.readme-ohmyfood.jpg)

## Mission
Integration of design model in HTML and CSS with SASS, animation creation

## Installation :
```bash
docker compose up -d
docker exec -it ohmyfood /bin/sh
npm install -g sass
sass --version
sass --watch sass:public/css
```
